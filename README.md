## 🚀 Project Showcase

| Project | What it does | Stack |
|---|---|---|
| 🚴🏾 **WayaSend** | Flutter delivery app with an InDrive-style bidding flow connecting clients and couriers | `Flutter` `Spring Boot` `Redis` `FCM` |
| ✂️ **Mucheka** | Digitising and formalising Zimbabwe's informal tailoring industry, from onboarding to order trust | `Flutter` `Spring Boot` `PostgreSQL` |
